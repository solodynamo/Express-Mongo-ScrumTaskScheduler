## Usage

    $ git clone https://github.com/solodynamo/Express-Mongo-ScrumTaskScheduler.git
    $ cd Express-Mongo-ScrumTaskScheduler
    $ npm install nodemon -g
    $ npm install
    $ npm start
    $ App running on 3000 port
